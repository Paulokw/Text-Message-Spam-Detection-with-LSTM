# Text-Message-Spam-Detection-with-LSTM
This is a NLP project which involves detection of spam in text messages.
We will be implementing Neural Networks by using LSTM model. We started off with text preprocessing by using regular expressions and wordnet lemmatization on the words. We then encoded the words and padded the sentences before applying embeddings so as to convert the encoded word to vectors. We then created our LSTM model before fitting our data and evaluating it using confusion matrix.
